# Data Cleaning & Preparation – Project 1

## Overview

This project was completed as part of the DecodeLabs Data Analytics Internship Program. The objective of the project was to clean and prepare a raw dataset by identifying missing values, checking duplicate records, correcting data formats, and ensuring overall data quality.

---

## Project Objective

The main goal of this project was to transform a raw dataset into a clean and analysis-ready dataset by:

* Identifying missing values
* Handling null entries
* Checking duplicate records
* Standardizing date formats
* Validating numeric data
* Cleaning text fields

---

## Tools Used

* Microsoft Excel
* Excel Functions (TRIM, Filters, Remove Duplicates, Format Cells)

---

## Data Cleaning Steps

### 1. Missing Value Handling

* Identified missing values in the CouponCode column.
* Replaced blank entries with "No Coupon".

### 2. Duplicate Record Verification

* Checked the entire dataset for duplicate rows.
* Verified that there were no duplicate Order IDs or Tracking Numbers.

### 3. Date Formatting

* Standardized all dates into a consistent format:

  YYYY-MM-DD

### 4. Numeric Data Validation

* Reviewed numeric columns such as Quantity, UnitPrice, ItemsInCart, and TotalPrice.
* Ensured proper formatting and consistency.

### 5. Text Cleaning

* Removed unnecessary spaces using the TRIM function.
* Verified text consistency across columns.

---

## Results

After completing the cleaning process:

* Missing values handled successfully
* Zero duplicate records
* Zero duplicate IDs
* Consistent date formatting
* Clean text fields
* Analysis-ready dataset
